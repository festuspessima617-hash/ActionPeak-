# ActionPeak Website

A responsive awareness and reporting platform focused on ending gender-based violence in Sierra Leone.

## Features
- Responsive Design
- Awareness Content
- Reporting Form
- Gallery
- Emergency Contacts
- Mobile Navigation

## Technologies Used
- HTML
- CSS
- JavaScript

## Deployment
The original static site is preserved, while the new ActionPeak NGO platform is implemented in `app/` as a React + Vite project.

## Getting started
1. Install dependencies: `npm install`
2. Start the frontend: `npm run dev`
3. Start the backend API: `npm run start-server`

## Platform structure
- `app/` contains the React application, Tailwind styles, and PWA support.
- `server/` contains the Express API, email notifications, and secure case routes.
